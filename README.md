DOS_RecycleBin_cleaner.bat is for Windows 95, Windows 98, and Windows ME (Millenium Edition).
W2K-Server03_RecycleBin_cleaner.bat is for Windows 2000, Windows XP, and Windows Server 2003.
WinVista-8.1_RecycleBin_cleaner.bat is mainly for Windows Vista, Windows 7, Windows 8, and Windows 8.1, but can also be used for Windows 10 and Windows 11.
Win10-11_RecycleBin_cleaner.bat is the native Windows 10 and Windows 11 app because it uses Powershell.

If you somehow got Powershell on an older version of windows and forced command prompt to open it using "Powershell.exe" then Win10-11_RecycleBin_cleaner.bat is the best version for you.
